# VSS CLI Mission (Lite)

VSS CLI is a comprehensive command-line interface that helps University of Toronto researchers, faculty, IT staff, and DevOps engineers manage ITS Private Cloud resources efficiently by providing intuitive VM lifecycle management, Infrastructure as Code capabilities, and AI-powered contextual assistance.

VSS CLI serves academic institutions and IT operations teams managing private cloud infrastructure for research computing and application deployment. Unlike generic cloud CLI tools, VSS CLI is purpose-built for ITS Private Cloud with deep integration of university-specific workflows, native MFA support, YAML-based Infrastructure as Code for version-controlled deployments, and an integrated GPT-powered assistant providing context-aware guidance directly within the CLI workflow.
